# 🌍 Terminal Weather Checker

A sleek Python tool to check live weather data directly from your terminal. Built on a Chromebook!

## 🚀 Features
* **Live Data:** Fetches real-time weather via the `wttr.in` API.
* **Interactive:** Search for any city in the world.
* **Modern Setup:** Uses Python virtual environments and `requests`.
